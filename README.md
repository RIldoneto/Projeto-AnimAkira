# AnimAkira

Um catálogo simples, bonito e funcional de animes feito com HTML, Tailwind CSS e JavaScript.

![Projeto-AnimAkira](https://via.placeholder.com/800x400/18181b/eab308?text=AnimAkira)

## ✨ Funcionalidades

- Design moderno, clean e totalmente responsivo
- Busca de animes em tempo real
- Modal com detalhes completos do anime
- Exibição de **Gêneros**
- Exibição de **Dificuldade** (Fácil, Médio, Difícil, Avançado)
- Nome exato do anime conforme a API
- Consumindo a API pública **Jikan** (MyAnimeList)

## 🛠 Tecnologias

- **HTML5** Semântico
- **Tailwind CSS** v4 (via CDN)
- **JavaScript** (Fetch API)
- **Jikan API**

text## 🚀 Como usar

1. Baixe os três arquivos
2. Abra o arquivo `index.html` no navegador
3. Pronto! Já pode buscar e explorar os animes

### Para hospedar online (GitHub Pages):

1. Crie um repositório no GitHub
2. Faça upload dos arquivos
3. Vá em **Settings > Pages** e ative o GitHub Pages na branch `main`

## 📌 Informações dos Animes

Cada anime exibe:
- Nome oficial (exato)
- Gêneros
- Dificuldade sugerida
- Nota (Score)
- Quantidade de episódios
- Sinopse
- Imagem de capa
